HOW TO RUN LOCALLY:
-------------------
1. Backend:
   cd backend
   pip install -r requirements.txt
   uvicorn server:app --reload --port 8001

2. Frontend (in new terminal):
   cd frontend
   yarn install
   yarn start

3. Open http://localhost:3000
